div dediğimiz etiket (division kısaltması) html etiketlerini çevreleyen, css'ler ile birlikte kullanılarak görsel tasarım oluşturulmasına yardımcı olan etikettir. Arayüzde görsel bir çıktı vermez.

id attribute: tüm elementlere id özelliği eklenebilir. name propertisi gibi bişi. bir id değeri sayfa içinde bir kez tanımlanabilir. (tekil, unique olması gerek)
class attribute: yine tüm elementlere class özelliği eklenebilir. ancak id gibi tekil olması zorunluluğu yoktur. birden fazla elemente aynı class adı tanımlanabilir. bir elemente hem class hem id değeri verilebilir.

style etikeri içinde CSS kodları yazarak tasarımda görsel düzenleme yapabiliriz. style tag, head tagi içinde tanımlanır. css kodlarını elementin içine de yazabiliriz.
